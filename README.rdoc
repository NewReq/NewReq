== What is NewReq?

NewReq is a simple and flexible tool for managing IT user access requests. But still in development :(
It is distributed under the GNU AFFERO General Public License (AGPL)


Key Features:

* Self Service access request

* Approval workflow

* LDAP integration (automation)

* Flexible entries

* Ruby on Rails Plataform
